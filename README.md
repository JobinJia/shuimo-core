# shuimo-core (水墨核心库)

A TypeScript library for procedural Chinese landscape painting generation.

## Standing On The Shoulders Of Giants 

[shanshui](https://github.com/LingDong-/shan-shui-inf)

[flower](https://github.com/LingDong-/nonflowers)

## Development

### Prerequisites

- Node.js >= 18.0.0
- pnpm >= 8.0.0

### Installation

```bash
pnpm install
```

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```
